# 🚀 クイックスタートガイド

価格転嫁支援AIアシスタントをすぐに起動する方法です。

## 最も簡単な方法（推奨）

### Windows

1. **`start_all.bat` をダブルクリック**
   - 依存パッケージのインストール（初回のみ）
   - バックエンドサーバーの起動
   - フロントエンドサーバーの起動
   - ブラウザが自動で開きます

2. **ブラウザで確認**
   - `http://localhost:5173` が自動で開きます
   - 開かない場合は手動でアクセスしてください

### 起動後の画面

- ウェルカムメッセージが表示されます
- チャット入力欄に質問を入力できます
- 例: 「原価計算のやり方を教えて」

## トラブルシューティング

### バックエンドが起動しない

```bash
# 依存パッケージを手動でインストール
pip install -r api/requirements.txt
pip install -r requirements.txt
```

### フロントエンドが起動しない

```bash
cd frontend
npm install
npm run dev
```

### ポートが使用中

- バックエンド: ポート8000が使用されている場合、他のアプリを終了してください
- フロントエンド: ポート5173が使用されている場合、Viteが自動で別のポートを使用します

### ブラウザが開かない

手動で以下のURLを開いてください：
- フロントエンド: http://localhost:5173
- バックエンドAPI: http://localhost:8000

## サーバーの停止方法

1. **統合起動スクリプトを使用した場合**
   - 各サーバーウィンドウで `Ctrl+C` を押す
   - または、ウィンドウを閉じる

2. **個別に起動した場合**
   - 各ターミナルで `Ctrl+C` を押す
