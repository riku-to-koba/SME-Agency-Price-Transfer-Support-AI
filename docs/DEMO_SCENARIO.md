# 価格転嫁支援AIアシスタント デモシナリオ（5分版）

## 📌 コンセプト: 「2つのモード × 自動判定 × 専門ツール連携」
このデモでは、以下の3つのシステムの特徴を、自然なストーリーの中で訴求します:

1. **2つのモード（よろず相談 ⇄ 価格転嫁専門）の自動切り替え**
2. **ステップ自動判定によるツールの最適選択**
3. **6つの専門ツールによる課題解決の流れ**

### デモの訴求ポイント
1. **Mode 1（よろず相談）**: 幅広い経営課題に対応、Web検索標準搭載
2. **Mode 2（価格転嫁専門）**: 14ステップのプレイブック、6つの専門ツール
3. **ステップ自動判定**: 会話内容から最適なステップとツールを自動選択
4. **専門ツールの連携**: 市場分析 → コスト試算 → 企業調査 → 文書生成の流れ

### 想定視聴者
- 価格交渉に悩む中小企業経営者・担当者
- 支援機関（商工会議所、よろず支援拠点）の担当者
- システムの技術的価値を評価したい人
