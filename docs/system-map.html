📋 Mode 1 Agent（子エージェント）

よろず相談エージェント

経営全般の幅広い相談に対応する汎用エージェント。
価格転嫁以外の課題（資金繰り、人材採用、マーケティング等）を傾聴し、適切なアドバイスを提供。
Web検索機能を標準搭載し、最新の市場動向や業界ニュース、統計データなどを取得可能。
ステートレス（軽量）：複雑な状態管理は不要。

💼 Mode 2 Agent（子エージェント）

価格転嫁専門エージェント

価格交渉に特化した専門家エージェント。6つの専門ツールを直接使用して、準備から実践まで徹底サポート。
14ステップ（CHECK 1-9 → STEP 1-5）は参考ガイドラインとして提供（順番の強制なし）。

🔄 Mode 2 Agentの動作
  1. ツール直接呼び出し：LLMがFunction Callingで6つの専門ツールを直接呼び出し
  2. 柔軟なツール選択：会話内容に応じてLLMが最適なツールを判断・実行
  3. 成果物管理：生成されたグラフ、文書、試算表などをセッションに保存
  4. シームレス実行：ユーザーは対話を続けるだけで、裏側で専門ツールが連携動作

例：「取引先の財務状況を知りたい」→ LLMがcompany_researchツールを直接呼び出し

🛠️ 6つの専門ツール（最小公倍数設計）

各ツールは単一の責務を持ち、LLMが判断・解釈を担当。組み合わせで高度な機能を実現

🔍
web_search
Web検索ツール
あらゆる外部情報取得の基盤。市場データ、企業情報、業界ニュースを検索し、構造化された結果を返す。
LLMの役割：検索結果から数値データを抽出・分析し、企業をランク付け（S/A/B/C）、交渉戦略を提案。
使用タイミング: CHECK 2（市場データ）, CHECK 6（取引先調査）, STEP 1（業界動向）
Tech: Web Search API
⚖️
search_knowledge_base
法務知識検索ツール
RAGで下請法、独禁法、価格転嫁ガイドライン、計算式を検索し、法的根拠を取得。
LLMの役割：取得した条文をユーザーの状況に適用し、違法性を判定、対抗話法を生成。
使用タイミング: CHECK 3（計算式参照）, CHECK 8（法令違反チェック）, 随時
Tech: RAG (S3 + Vector Search), FAISS/Pinecone
💰
calculate_cost_impact
コスト影響試算ツール
複雑な数値計算・シミュレーションを実行し、松竹梅プランを自動生成。
LLMの役割：計算結果を解釈・説明し、どのシナリオを推奨すべきかを判断。
使用タイミング: CHECK 3, CHECK 9（最重要）
Tech: NumPy, Pandas, Excel出力
📊
generate_chart
グラフ生成ツール
データを受け取りグラフ画像を生成し、S3に保存してURLを返す。
LLMの役割：web_searchやcalculate_cost_impactの結果からデータを抽出し、適切なグラフタイプを選択、解釈。
使用タイミング: CHECK 2（市場データ）, CHECK 9（コスト試算）, STEP 1（交渉資料）
Tech: Matplotlib, Seaborn, S3
📄
generate_document
文書自動生成ツール
テンプレート処理を実行し、PDF/Excel/Wordファイルを生成・S3保存。
LLMの役割：文書の内容・論理構成を作成し、適切なトーンを判断、法的根拠の引用箇所を指定。
使用タイミング: CHECK 1, 4, 5, STEP 3, 4, 5など（全ステップで活用）
Tech: Jinja2, python-docx, openpyxl, PDF
🎭
simulate_negotiation
交渉シミュレーションツール
相手役のペルソナを管理し、スコアリング機能を提供。
LLMの役割：ペルソナに応じた相手役の発言を生成し、ユーザーの交渉戦略を評価・改善提案。
使用タイミング: CHECK 7（強みの言語化後）, STEP 4（交渉本番前）
Tech: LLM (Multi-turn), スコアリングエンジン
